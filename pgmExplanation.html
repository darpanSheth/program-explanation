Age category programe explanation

Explanation for the Javacript code fragment given below :-
Basically, this program gives us the category in which a person fall on the basis of their age. Firstly, It defines a variable checkAgeCategory and asks the user for some input.
Than, it converts the String to a Number using numberFormat since the user input from prompt window is always in the form of "String"
Based on user's input, it will give the output defining the person's category by using if, else-if, else conditional statements that varies output for different conditions.

If the checkAgeCategory is (≤12), output will be "You are still a child!"
If the checkAgeCategory is (>12 & ≤19), output will be "You are a teenager!"
If the checkAgeCategory is (>19 & <60), output will be "You are an adult!"
If the checkAgeCategory is (≥60), output will be "You are a senior adult!"

Javascript code :


    let checkAgeCategory = prompt("Please enter you age");   //Prompts user for the age input
    checkAgeCategory = Number(checkAgeCategory);             //Converts String input to a Number
    
    if(checkAgeCategory >= 60){
        alert("You are a senior adult");
    }
    else if(checkAgeCategory > 19){
        alert("You are an adult!");
    }
    else if(checkAgeCategory > 12){
        alert("You are a teenager");
    }
    else{
        alert("You are still a child!");
    }

Please note: While using if, else-if statements, the programe will terminate if the first condition is true.
That is why we have checked the higher age condition first in our code and not the lower age condition.

For instance- If we checked the (checkAgeCategory > 12) condition first, output for teenager would only be displayed.
And our programe will not take the fact into consideration that this person could also be an adult or a senior adult.
So, sometimes the programming logic seems to be simple but it is not often as easy as it seems.
